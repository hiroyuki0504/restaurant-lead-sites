About Kura

女将の手から、宮古の記憶が一皿ずつ。

繁華街から少し離れた久貝の静かな場所で、蔵は宮古島の家庭料理を守り続けています。派手な演出よりも、島の野菜、地魚、宮古味噌、受け継いだ味。会話と料理を急がず楽しむための、小さな食事処です。
